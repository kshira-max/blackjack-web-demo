# ブラックジャックゲーム

このプロジェクトは、ブラックジャックゲームを実装したものです。プレイヤーはディーラーと対戦し、21に近い手札を目指します。

## プロジェクト構成

- `src/index.ts`: アプリケーションのエントリーポイント。ゲームの初期化や開始処理を行います。
- `src/game/card.ts`: `Card`クラスを定義し、カードのスートや値を管理します。
- `src/game/deck.ts`: `Deck`クラスを定義し、カードのデッキを管理します。
- `src/game/hand.ts`: `Hand`クラスを定義し、プレイヤーまたはディーラーの手札を管理します。
- `src/game/game.ts`: `Game`クラスを定義し、ゲームの進行を管理します。
- `src/ui/cli.ts`: CLI用のUIを提供し、ユーザーからの入力を受け取ります。
- `src/ui/web.ts`: Web用のUIを提供し、ブラウザでのインタラクションを管理します。
- `src/services/dealer.ts`: `Dealer`クラスを定義し、ディーラーの行動を管理します。
- `src/utils/helpers.ts`: ゲーム内で使用されるヘルパー関数を定義します。
- `src/types/index.ts`: ゲームで使用される型やインターフェースを定義します。

## テスト

- `tests/game.test.ts`: ゲームロジックに関するユニットテストを含みます。
- `tests/utils.test.ts`: ヘルパー関数に関するユニットテストを含みます。

## 実行方法

1. リポジトリをクローンします。
2. 依存関係をインストールします。
   ```
   npm install
   ```
3. ゲームを実行します。
   ```
   npm start
   ```
